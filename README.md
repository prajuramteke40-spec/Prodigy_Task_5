# Prodigy Infotech: Machine Learning - Task 05

## 📸 Object Detection in Images

This repository contains the implementation of Task 05 for the Prodigy Infotech Machine Learning Internship. The objective of this project is to implement an object detection model capable of identifying and locating multiple objects within a single image, drawing accurate bounding boxes around them.

### 📋 Project Overview
Object detection goes a step further than standard image classification by not only predicting *what* is in an image, but *where* it is located. In this project, we utilize a pre-trained deep learning object detection model (such as YOLO or SSD) to process images, recognize various everyday objects, and label them with confidence scores.

This notebook demonstrates the complete workflow: loading the image, passing it through the neural network, extracting the bounding box coordinates and class labels, and rendering the final annotated image.

### 🛠️ Technologies & Libraries Used
* **Python 3.x:** Core programming language.
* **OpenCV / PIL:** For computer vision tasks, image reading, and drawing bounding boxes.
* **Deep Learning Frameworks:** (e.g., PyTorch, TensorFlow, or Ultralytics YOLO) for loading and running the object detection model.
* **Matplotlib:** For rendering the final output images inline.
